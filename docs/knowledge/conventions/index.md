# Conventions

リポジトリ横断の取り決めを置く。`biome.json` / `tsconfig.json` / `astro.config.mjs` / `panda.config.ts` 等の設定ファイルに既に書かれている値の再記述はしない。設定値では表現しきれない **判断基準と例外処理の運用** を記録する場所。

`AGENTS.md` には人間と AI エージェントが最初に流し読みする短いサマリが載っている。このディレクトリのドキュメントは、そのルールの **背景の why** を膨らませ、エッジケースの扱い方を明確にする。
